# Nordic EnergyCare — GTM-Controlled Energy Support Chatbot Platform

A professional full-stack energy customer-service portal built for junior full-stack developer interview preparation.

## Goal

This project demonstrates a realistic customer-service platform where chatbot loading, opening, and tracking can be controlled through Google Tag Manager and custom frontend events.

## Planned Tech Stack

### Frontend
- React + Vite
- JavaScript
- Tailwind CSS
- React Router
- Google Tag Manager
- Botpress Webchat
- dataLayer events

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT authentication
- bcrypt
- express-validator
- CORS
- dotenv

### Testing
- Postman
- Cypress
- Manual testing checklist
- Browser DevTools
- GTM Preview Mode

### DevOps
- Docker
- Docker Compose
- GitHub Actions basic CI
